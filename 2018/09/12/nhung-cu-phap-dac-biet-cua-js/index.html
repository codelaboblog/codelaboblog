Những cú pháp đặc biệt của Javascript

2018-09-12

Trong bài viết này, chúng ta sẽ nói về những thứ mà ai cũng vậy “Nhưng JS thì không”

Lệnh gán có thể ở khắp mọi nơi

Thông thường những ngôn ngữ khác sẽ cố định lệnh gán phải nằm riêng biệt trên 1 dòng lệnh, nhưng JS thì không

if (a = 1) { }

const a = window.myCache || (window.myCache = {});
‍‍‍ ‍‍‍ ‍‍‍ ‍‍‍ ‍‍‍ ‍‍‍‍‍‍
functionCall(a = 100);

‍‍‍

Rút gọn điều kiện true/false

JS xem những biểu thức null, undefined, chuỗi rỗng, số 0 tương đương với false
Và xem số khác 0, chuỗi không rỗng, object bất kỳ tương đương true

if(a === 0
  || a === null
  || a === undefined
  || a === ''){

}
// có thể rút gọn thành
if(!a){ }

Biểu thức && và || không trả về true/false

Các ngôn ngữ khác mặc định rằng kết hợp && hoặc || sẽ luôn là true/false nhưng JS thì không

const array = [1, 2, 3];
const length = itemChanged && array.length;
// nếu itemChanged là false thì length = false;
// nếu itemChanged là true thì length là array.length;

function plugin(options) {
  options = options || {};
  // nếu options chưa khởi tạo thì gán options bằng {}
  // ngược lại thì vẫn dùng lại giá trị options
}

Dùng bitwise operator để nhân hoặc chia

100 << 1 => 200
100 << 2 => 400
100 >> 1 => 50
100 >> 2 => 25

Dùng switch thay thế nhiều if phức tạp

Một số ngôn ngữ chỉ cho phép case chứa những primitive value, nhưng JS thì không

if(a.length > 0){
  // do something 1
}
else if(b.call()){
  // do something 2
}
else {
  // do something 3
}
// có thể chuyển thành
switch(true){
  case a.length > 0:
    // do something 1
  case b.call():
    // do something 2
  default:
    // do something 3
}
